# Password Generator

## Overview
This project is a GUI-based Password Generator developed using Python and Tkinter.  
It helps users generate strong and secure passwords based on selected options.

## Objective
- Generate strong passwords
- Allow users to choose password length
- Include or exclude character types

## Technologies Used
- Python
- Tkinter

## Features
- Custom password length
- Uppercase, lowercase, numbers, and symbols
- Option to exclude specific characters
- Copy password to clipboard

## Output
The application generates a secure password instantly and allows users to copy it easily.

## Conclusion
This project demonstrates Python GUI development and secure password generation.
